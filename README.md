# PixivCrawler
Crawling pictures from daily ranking list of pixiv.

根据日期构造排行榜URL -> 获取当日排行榜中的所有图片ID -> 根据图片ID构造图片页URL -> 保存原图

P.S. cookies.txt required.

Login first, then get the cookies and copy them to cookies.txt.
